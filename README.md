專案白皮書：智慧魔術方塊解題 App (Smart Cube Solver)
專案背景：國立雲林科技大學 (Yuntech) 電腦視覺與互動應用專案文件更新日期：2026 年 5 月

專案願景：把無解的挫折，變成充滿成就感的魔法
每個人家裡的抽屜，幾乎都有一個轉亂後就再也沒復原過的魔術方塊。這款「智慧魔術方塊解題 App」的誕生，就是為了打破這個僵局。

我們結合了行動裝置的相機鏡頭、即時的電腦視覺（Computer Vision）影像處理，以及 3D 空間指引技術。使用者不再需要死背複雜的公式，只要跟著手機畫面「拍幾張照，跟著動畫轉」，就能在幾分鐘內親手解開魔術方塊。這不僅是一個工具 App，更是一個展現演算法與擴增實境（AR）如何完美結合的展示舞台。

核心體驗：四步完成的奇幻旅程
我們將複雜的技術隱藏在極度簡潔的四個步驟背後：

掃描 (Scan) - 讓手機看懂方塊：
使用者打開 App，將鏡頭對準實體方塊。畫面上的九宮格輔助線會自動捕捉顏色。系統會引導使用者翻轉方塊，將六個面依序拍攝完畢。

驗證 (Validate) - 確保所見即所得：
系統將掃描結果攤平為一張 2D 展開圖。使用者一眼就能確認系統辨識的顏色是否精準，如果有任何色塊因為反光被誤判，只需點擊一下即可手動修正。

計算 (Calculate) - 毫秒級的大腦運算：
確認無誤的顏色矩陣，會被瞬間送入核心演算法（例如著名的 Kociemba 演算法）。系統在不到 1 秒的時間內，就能算出 20 幾步以內的最佳還原路徑。

指引 (Guide) - 傻瓜式的 3D 教學：
這也是整個 App 的高光時刻。畫面上會出現一個 3D 虛擬方塊，使用者只要按下「播放」，3D 箭頭就會一步步示範該怎麼轉。可以隨時暫停、重播，確保手上的實體方塊與畫面完全同步，直到六面全部還原。

藏在幕後的技術挑戰與解法
NotebookLM 主持人請注意，這三個技術亮點是本專案的精華，展示了我們如何用工程思維解決現實世界的物理干擾：

技術亮點一：對抗光線魔王的 HSV 色彩空間
魔術方塊的塑膠表面非常容易反光，如果用傳統的 RGB 來辨識，白光反光很容易讓系統瞎掉。為此，我們將影像處理轉換為 HSV 色彩空間。透過分離「色相 (Hue)」與「亮度 (Value)」，即使在陰影或黃光下，系統依然能精準分辨到底那是橘色還是紅色。

技術亮點二：同調性檢查 (Parity Check) 防呆機制
有時候方塊曾經摔壞、被亂裝回去，這在數學上會形成「死局」。為了不讓演算法陷入死迴圈，我們在計算前加入了嚴格的數學同調性檢查。如果這顆方塊根本無解，系統會第一時間溫柔地警告使用者：「你的方塊可能曾經被拆裝過喔！」

技術亮點三：零認知負擔的 3D 空間對位
「左邊」跟「右邊」在轉動立體方塊時很容易搞混。為了解決空間迷失，我們在 3D 動畫中導入了「中心塊定錨」設計，永遠提示使用者「紅色中心塊朝向自己」，並加上高對比的 3D 箭頭，讓空間認知負擔降到最低。

產品開發的下一步 (Technical Spike)
我們目前正處於「技術突擊（Technical Spike）」的快速驗證階段。我們暫時不花時間雕琢華麗的 UI，而是優先確保骨幹暢通：
從「手機鏡頭擷取影像」 ➡️ 「成功轉換出 54 個顏色陣列」 ➡️ 「演算法算出正確解法」 ➡️ 「Unity 裡的幾何方塊能照著指令轉動」。

一旦這個核心迴圈被驗證成功，我們就會將規格交付給 UI/UX 設計師與 3D 美術團隊，加入充滿科技感的掃描特效與細膩的材質光影，將這個硬核的技術專案，包裝成每個人都愛不釋手的日常魔法。
